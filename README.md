# 📊 EDA Insights App

> An automated Exploratory Data Analysis web application — upload any dataset and instantly get statistics, visualisations, and data quality reports. No coding required.

---

## 🔍 What This App Does

Most data projects spend 60–80% of time on EDA before any real analysis begins. This app automates that process.

Upload a CSV file → get a full data profile in seconds.

**Features:**
- ✅ Automatic descriptive statistics (mean, median, std, min, max)
- ✅ Missing value detection and report
- ✅ Data type identification
- ✅ Interactive charts and distributions (Plotly)
- ✅ Correlation heatmap
- ✅ Clean, non-technical output — built for business stakeholders

## 🚀 How To Run Locally

**1. Clone the repo**
```bash
git clone https://github.com/nadzirahzuber/Eda-App.git
cd Eda-App
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Run the app**
```bash
streamlit run app.py
```

**4. Open in browser**
```
http://localhost:8501
```
